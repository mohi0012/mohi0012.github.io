Learning about HTML Tags

  • Images: a html tag that allows for images to appear on the page
  • Footer: The ending of a page's html tag that shows the person who published the website, the year it was published, and any information that pertains to the website's holder
  • Headers: When placing a title to the page, a header must be placed to draw attention to the title, as it it is larger in size than the paragraph
  • Body: This html tag is placed for the body of the website's content, which involves any headers, links, paragraphs, quotes, etc.
  • Links: To travel to a different page or website, links are placed on the website to access those points on the web
